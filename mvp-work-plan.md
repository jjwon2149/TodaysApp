# Duo-Style Photo Habit App MVP Work Plan

## 0. 문서 목적

이 문서는 `하루 한 장 사진 기록 앱`의 MVP를 실제로 구현하기 위한 작업 계획서다.  
목표는 아이디어를 좋은 문장으로 정리하는 것이 아니라, `바로 만들 수 있는 단위`로 쪼개는 것이다.

이 제품은 단순한 사진 앱이 아니라, `매일 돌아오게 만드는 습관 구조`를 가진 개인 기록 앱이어야 한다.  
핵심은 다음 두 가지다.

1. 사용자가 `10초 안에 오늘 기록을 시작`할 수 있어야 한다.
2. 사용자가 `7일 이상 다시 돌아오게 만드는 루프`가 있어야 한다.

---

## 1. 이번 작업의 최종 목표

### 1.1 MVP 목표

아래 흐름이 안정적으로 동작하는 iOS 앱을 만든다.

1. 첫 실행
2. 온보딩
3. 알림 설정 또는 건너뛰기
4. 홈 진입
5. 오늘 사진 촬영 또는 선택
6. 메모/감정 입력
7. 저장
8. streak 반영
9. 캘린더 반영
10. 다음 날 다시 돌아오기

### 1.2 MVP 성공 기준

1. 앱을 설치한 사용자가 첫날 기록을 쉽게 완료할 수 있어야 한다.
2. 기록 데이터가 로컬에 안정적으로 저장되어야 한다.
3. streak 계산이 예외 없이 일관되게 동작해야 한다.
4. 캘린더에서 쌓이는 느낌이 보여야 한다.
5. 알림 없이도 앱 핵심 기능이 완전하게 동작해야 한다.

## 2. 제품 방향 원칙

### 2.1 반드시 지킬 원칙

1. `BeReal`처럼 보이면 안 된다.
2. 랜덤 실시간 인증 구조는 넣지 않는다.
3. 핵심은 `친구 공유`가 아니라 `개인 기록 루프`다.
4. 사용자를 죄책감으로 붙잡지 않는다.
5. 스트릭은 보상 장치이지 벌점 장치가 아니다.
6. 초기 앱은 작아야 하고, 루프는 강해야 한다.

### 2.2 실행 원칙

1. 먼저 `핵심 루프`를 만든다.
2. 그다음 `누적 시각화`를 붙인다.
3. 그다음 `보상 구조`를 붙인다.
4. 마지막으로 `수익화/확장성`을 붙인다.

## 7. 화면별 상세 작업 분해

## 7.1 Onboarding

### 목표

사용자가 앱의 정체성을 20~30초 안에 이해하게 만든다.

### 구현 항목

1. 페이지 1: 하루 한 장 기록
2. 페이지 2: streak와 캘린더 가치
3. 페이지 3: private-first 설명
4. 시작 CTA
5. 건너뛰기 CTA
6. 온보딩 완료 상태 저장

### 필요한 상태

1. `onboardingCompleted`
2. 현재 페이지 index

### 완료 조건

1. 앱 재실행 시 온보딩 중복 노출되지 않아야 한다.
2. 건너뛰기와 시작하기가 모두 정상 동작해야 한다.

## 15. QA 체크리스트

## 15.1 기록 플로우

1. 첫 기록이 정상 저장되는가
2. 당일 재저장이 정상 교체되는가
3. 사진 없이 저장이 막히는가
4. 메모 없이 저장이 가능한가
5. 앱 재실행 후 기록이 유지되는가

## 15.2 streak 플로우

1. 오늘 기록 시 streak가 증가하는가
2. 하루 건너뛰면 reset 또는 freeze가 적용되는가
3. 과거 기록 삭제 시 재계산되는가
4. 자정 이후 첫 진입 시 상태가 맞는가

## 15.3 권한 플로우

1. 카메라 거부 시 대안 경로가 보이는가
2. 앨범 거부 시 안내가 적절한가
3. 알림 거부 시 앱 사용이 막히지 않는가

## 15.4 캘린더/상세

1. 월 이동이 안정적인가
2. 썸네일 누락 시 깨지지 않는가
3. 상세에서 수정/삭제가 정책대로 동작하는가

## 18. 권장 실행 결론

이 앱은 기능이 많은 앱으로 시작하면 안 된다.  
처음부터 잘 만들어야 하는 것은 `핵심 루프`와 `습관 메커니즘`이다.

가장 먼저 완성해야 할 것은 아래다.

1. 오늘 기록할 수 있는가
2. 기록이 저장되는가
3. streak가 맞게 계산되는가
4. 캘린더에 쌓이는가
5. 다음 날 다시 돌아올 이유가 있는가

이 다섯 가지가 닫히면, 그다음부터 배지, 프리미엄, 회고, 위젯, 공유를 붙여도 된다.

현재 단계에서의 가장 현실적인 실행 순서는 다음과 같다.

1. `Home`
2. `Capture`
3. `EntryEditor`
4. `Save`
5. `Streak`
6. `Calendar`
7. `Detail`
8. `Notification`
9. `Badge`
10. `Premium Shell`
